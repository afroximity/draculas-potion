# Changelog

All notable changes to "Dracula's Potion" will be documented in this file.

## [1.0.0] - 2026-02-03

### Added
- Initial release
- Ultra-dark background (#080B0E) for maximum contrast
- Complete UI theming with 100+ color definitions
- Enhanced Elixir syntax highlighting:
  - Atoms and atom colons (cyan)
  - Modules and structs (purple/cyan)
  - Module attributes (orange)
  - Operators and keywords (pink)
  - Strings and sigils (yellow)
  - Lambda captures (orange)
  - HEEx/EEx embedded tags (purple)
  - Typespecs and guards
- Semantic highlighting support
- Rainbow bracket colorization
- Full terminal color support
